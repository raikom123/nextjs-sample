# Next.jsの実装サンプル

Next.js公式の[チュートリアル](https://nextjs.org/learn/foundations/about-nextjs)を実装しました。

## 内容

チュートリアルを通じてNext.jsの基本的な動作について学びました。

## コマンド

パッケージのインストール
```shell
npm install
```

アプリの起動
```shell
npm run dev
```

## 課題

一般的なシステムで利用されると思われる機能として下記を実装する。

- 入力チェック
- 認証・認可
- DBアクセス
- dockerでNext.jsを起動する
